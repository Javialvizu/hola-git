#### **Permisos y propietarios**

\- rw- r-- r--

  │   │   └─ otros (o)

  │   └──── grupo (g)

  └──────── usuario dueño (u)



-usuario(dueño): rw-(leer,escribir)

-grupo: r--(solo leer)

-otros: r--(solo leer)



**Forma básica**

chmod -x archivo        # quitar permiso de ejecución

chmod +x archivo        # dar permiso de ejecución

chmod -r archivo | chmod -r archivo

chmod -w archivo | chmod -w archivo



**Permisos por categorías**

chmod u+r archivo    # usuario

chmod g+w archivo    # grupo

chmod o-r archivo    # otros

chmod go-w archivo   # grupo y otros

chmod ugo+r archivo  # todos

chmod go-rwx archivo # grupo y otros (todo)





**Realizar backups de un solo archivo**

cp archivo.txt ../BACKUPS/archivo\_backup.txt



#### **Usuarios y grupos**

whoami                 # ver tu usuario actual

id                     # ver UID, GID y grupos

sudo adduser nombre    # crear usuario (completo, con home)

sudo passwd nombre     # cambiar contraseña

sudo deluser nombre    # eliminar usuario

sudo deluser --remove-home nombre   # eliminar usuario + carpeta personal

sudo chown usuario\_nuevo nombre\_archivo #cambiar dueño de archivo



groups                 # ver grupos donde perteneces

sudo addgroup grupo    # crear grupo

sudo usermod -aG grupo usuario   # agregar usuario a grupo

#### **-----GESTIÓN DE SOFTWARE EN LINUX (APT)-----**



**APT (Advanced Package Tool) es el sistema de gestión de software en Ubuntu.**

**Sirve para:**

\- instalar programas

\- actualizar el sistema

\- eliminar software

\- buscar paquetes

\- mantener el sistema seguro y estable



---

**Actualizar el sistema**

---



sudo apt update          # actualizar la lista de paquetes disponibles

sudo apt upgrade         # actualizar los paquetes instalados

sudo apt full-upgrade    # actualizar sistema permitiendo cambios de dependencias



---

**Instalar paquetes**

---



sudo apt install nombre\_paquete     # instalar un programa



Ejemplos:

sudo apt install htop                # monitor de procesos avanzado

sudo apt install neofetch            # información del sistema

sudo apt install nmap                # escaneo de red

sudo apt install traceroute          # ver ruta de red

sudo apt install tree                # ver árbol de carpetas



---

**Ejecutar programas instalados**

---



htop        # abrir monitor de procesos

neofetch    # mostrar información del sistema

tree        # mostrar estructura de carpetas



---

**Buscar información de paquetes**

---



apt search nombre          # buscar paquetes disponibles

apt show nombre            # ver información detallada del paquete



Ejemplo:

apt search nmap

apt show nmap



---

**Desinstalar paquetes**

---



sudo apt remove nombre     # desinstalar programa (mantiene configuración)

sudo apt purge nombre      # desinstalar + borrar configuración

sudo apt autoremove        # eliminar dependencias que ya no se usan



---

**Reparar errores de APT**

---



sudo dpkg --configure -a   # reparar instalación interrumpida

sudo apt --fix-broken install   # arreglar dependencias rotas



---

**Limpiar el sistema**

---



sudo apt clean             # borrar archivos .deb descargados

sudo apt autoclean         # limpiar paquetes antiguos

sudo apt autoremove        # eliminar dependencias innecesarias



---

**Buenas prácticas**

---



\- Siempre ejecutar "sudo apt update" antes de instalar

\- No cerrar la terminal mientras APT trabaja

\- Usar "purge" si no quieres dejar rastros

\- Mantener el sistema actualizado regularmente
